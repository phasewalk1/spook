# Install

> [!TIP]
> **spook** uses [uv](https://docs.astral.sh/uv/getting-started/installation/) for building/packaging. Make sure you have it installed on your system.

**Clone the Repository**
```shell
git clone https://github.com/phasewalk1/spook
cd spook
```

**Build and Install**
```shell
uv tool install .
```

**Happy Hunting**!
```shell
spook -h
```
